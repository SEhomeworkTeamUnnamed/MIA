	\section{测试说明} % (fold)

		\subsection{功能测试} % (fold)
软件功能主要包括使用函数、添加函数和搜索函数这三个功能，于是按照这三个功能进行单元测试。另外，还要对整个软件的外观可操作部分点击功能进行集成测试。
		
		% subsection 功能测试 (end)

		\subsection{压力测试} % (fold)
因为根据软件功能需要，并不需要生成很大的Mathematica的代码，所以整个软件并不面临极大的压力测试。
		
		% subsection 压力测试 (end)
		
	% section 测试说明 (end)
